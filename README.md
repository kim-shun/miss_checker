# 目指した課題解決
 解決を目指していた課題は「仕事のミスをなくす」ことです。そしてミスを次に生かす導線として「ミスの原因を考える手間を省く」フォーマットが必要だと考えました。まずはユーザーが「選択肢から選ぶ」という単純な作業をするだけでミスの原因を浮き彫りにできるようにしたいです。
